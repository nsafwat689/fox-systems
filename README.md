# Fox Systems - IT Solutions and Services

A modern, responsive website for Fox Systems showcasing IT solutions and services.

## Features

- Responsive design with modern UI/UX
- Professional service showcase
- Contact forms and interactive elements
- Optimized performance for fast loading

## Technologies Used

- React 19
- TypeScript
- Tailwind CSS
- Vite
- Radix UI Components
- Framer Motion for animations

## Deployment

This website is built and optimized for production deployment using Vite.
